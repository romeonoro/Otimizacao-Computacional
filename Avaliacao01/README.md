Construa um código (em qualquer linguagem de programação) que receba uma série de equações formando um sistema.
A única regra desse sistema é que deve possuir o número de equações igual ao número de variáveis.
O programa deve resolver a diagonalização (tornar os valores da diagonal iguais a 1 e os valores acima e abaixo
da diagonal iguais a zero), mostrando a matriz de valores passo a passo na sua resolução.
O programa deve ser entregue no git até o dia 23/03 (submetido NESSA PASTA com nome diag-[NOME].py) e apresentado em aula.
Esse trabalho vale 5.0 pontos na primeira nota.
